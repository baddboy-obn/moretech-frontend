# Moretech FrontEnd
***
## Start in dev
```bash
#Install modules
yarn install
#Start app
yarn start
```
## Build in prod
```bash
#Install modules
yarn install
#Build app
yarn build-prod
```
## Build in dev
```bash
#Install modules
yarn install
#Build app
yarn build
```
## Lint application
```bash
#Install modules
yarn install
#Build app
yarn lint
```
